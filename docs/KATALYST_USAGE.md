# Katalyst Usage

Rileks uses AppWorkZ Katalyst as a lightweight project context method. The repo does not currently contain a plugin system, so Katalyst is registered through `.katalyst/README.md` instead of an executable plugin install.

## Locked Scope

This phase is a clickable, Vercel-deployable mockup only: landing page, chat simulator, dashboard, and five module pages.

## Intentionally Not Implemented

No backend, database, auth, WhatsApp, Telegram, Supabase, Neon, Cloudflare database, payments, OCR, AI assistant, payroll, or real accounting is included.

## How Future Agents Continue

1. Read `.katalyst/README.md` before changing scope.
2. Preserve useful prototype work.
3. Isolate any new phase before implementation.
4. Add integrations behind clear interfaces and only after explicit request.
5. Keep the Vercel mockup deployable at every step.
